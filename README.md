# Backend

## Реализованная функциональность

* Настроены мок запросы
* Чат на WebSocket
* Granat - голосовой помощник
* Auto deploy

## Стек
### Frontend
* React
* Typescript
* Docker, Docker-compose

### Backend
* Python, FastAPI
* PostgreSQL, SQLAlchemy
* Websockets
* Docker, Docker-compose
